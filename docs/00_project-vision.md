# Project Vision

The Archive is an Android-first free-to-play mobile extraction horror game built in Unity 6+.

The player is not the character. The player is an external human connecting into an archive-like hostile system through an avatar. The game should feel psychological, tense, systemic, and deliberate. It should not drift into arcade-shooter language, pacing, or reward loops.

The core experience should support extraction-session play: enter a hostile space, gather or recover valuable information and resources, manage risk, and extract before the system overwhelms the avatar. Future systems should remain compatible with bot-driven simulation and eventual multiplayer transport, but the initial workspace should stay focused and deterministic.

The project favors data-driven content, prefab-based authoring, mobile-readable silhouettes, and clean separation between gameplay logic and presentation.
