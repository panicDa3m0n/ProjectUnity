# The Archive

The Archive is an Android-first free-to-play mobile extraction horror game built with Unity 6+. The player is an external human connecting into a hostile archive-like system through an avatar, with a psychological, tense, systemic tone rather than arcade-shooter pacing.

## Setup Overview

- Unity project location: `unity/TheArchive/`
- Engine target: Unity 6+
- Platform target: Android first
- Rendering target: URP mobile
- IDE: Visual Studio Code with Codex
- Version control: Git with Git LFS for binary assets

Unity has not been installed and the Unity project has not been created yet. This repository currently contains only the initial workspace scaffold, documentation, Git rules, and Codex workflow configuration.

## Recommended Next Step

Initialize Git and Git LFS, then create the Unity project under `unity/TheArchive/` using Unity 6+ with the URP mobile template or an equivalent mobile URP setup.
